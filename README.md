## 🎯 About The Project

**SAVATSYA GAU SAMVARDHAN** is a digital platform created for a small-scale family-run business dedicated to preserving traditional methods of creating pure and natural products. This e-commerce website brings their authentic incense sticks and A2 cow ghee to customers worldwide, combining ancient Vedic wisdom with modern web technology.

### 🌟 Key Features

- 🛒 **E-commerce Platform** - Complete shopping experience with cart, wishlist, and checkout
- 🕯️ **Handcrafted Incense Sticks** - Traditional dhoop made with natural ingredients
- 🥛 **Pure A2 Cow Ghee** - Authentic ghee from our gau samvardhan (cow protection) center
- 🎨 **Beautiful UI/UX** - Elegant design reflecting traditional Indian aesthetics
- 📱 **Responsive Design** - Seamless experience across all devices
- ⚡ **Fast Performance** - Built with modern web technologies for optimal speed
- 🔐 **Secure** - Safe and secure payment processing
- 🌍 **Accessible** - WCAG compliant for inclusivity

## 🚀 Getting Started

### 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v18 or higher) - [Download here](https://nodejs.org/)
- **npm** or **yarn** package manager
- **Git** - [Download here](https://git-scm.com/)

### 🔧 Installation

Follow these simple steps to get the project running locally:

#### 🔀 **Option 1: Local Development**

```bash
# 1️⃣ Clone the repository
git clone https://github.com/Ved-V-Joshi/Savatsya-Gau-Samvardhan.git

# 2️⃣ Navigate to the project directory
cd Savatsya-Gau-Samvardhan

# 3️⃣ Go to the frontend directory
cd frontend

# 4️⃣ Install dependencies
npm install

# 5️⃣ Start the development server
npm run dev

# 🎉 Open your browser and visit http://localhost:5173
```

#### ✏️ **Option 2: Edit on GitHub**

1. 📂 Navigate to any file in the repository
2. ✏️ Click the "Edit" button (pencil icon)
3. 🔄 Make your changes and commit them
4. 🚀 Changes will be reflected automatically

#### ☁️ **Option 3: GitHub Codespaces**

1. 🟢 Click the "Code" button on the repository page
2. 📑 Select the "Codespaces" tab
3. ➕ Click "New codespace"
4. 💻 Edit files directly in the cloud-based IDE
5. 💾 Commit and push your changes

### 🏗️ Build for Production

```bash
# Build the project
npm run build

# Preview the production build
npm run preview
```
